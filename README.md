Zostal zaimplementowany oscylator harmoniczny bez tlumienia.
Program ulatwia zaimplementowanie innych modeli symulacji.
By uruchomic program trzeba stworzyc klase Simulation i wywolac jej metode run() - dwie przykladowe symulacje sa na koncu pliku.
Program zwraca wykres x(t), liste stanow, liste statystyk stanow i statystyki koncowe.
